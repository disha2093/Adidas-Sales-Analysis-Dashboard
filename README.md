# Adidas-Sales-Analysis-Dashboard
Adidas sales dashboard built with Power BI, showcasing advanced visualization, trend analysis, and performance metrics. Demonstrates practical skills in data analysis, BI, and business insights.



🔍 Overview

This project presents an interactive sales dashboard built using Power BI, analyzing performance data for Adidas. The goal is to uncover key business insights across sales, profit, regions, and product categories.


📈 Key Metrics

Total Sales: $899.9M

Total Profit: $332.1M

Units Sold: 2.5M

Average Price per Unit: $45.2

Average Margin: 42.3%

📊 Dashboard Features

   📅 Sales Trends

  Monthly sales performance visualization

  Peak sales observed around mid-year (July–August)

  🌍 Regional Analysis
  
  Sales distribution across regions (West, Northeast, Southeast, South, Midwest)
  
  Helps identify high-performing areas
  👟 Product Insights
  
Top-performing categories:

Men’s Street Footwear

Women’s Apparel

Enables product-level decision-making

🏬 Retailer Performance

Comparison of sales across retailers like:

West Gear

Foot Locker

Sports Direct

🛠️ Tools & Technologies

Power BI

Data Visualization

Business Intelligence Techniques

🚀 Key Insights

Strong seasonal trends in sales

Certain regions dominate revenue contribution

A few retailers generate the majority of sales

📌 Conclusion


This dashboard helps businesses make data-driven decisions by identifying trends, top products, and key revenue drivers.
